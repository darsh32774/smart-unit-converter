# 📱 Smart Unit Converter

A modular React Native mobile application that supports multiple unit conversions including distance, weight, temperature, currency, and number systems.

---

## 🚀 Features

- 🔢 Distance, Weight, Temperature, Currency, and Base conversions  
- 🧩 Modular architecture (feature-based structure)  
- 🌗 Light / Dark theme support  
- 🔁 Real-time currency conversion via external API  
- 🎯 Reusable UI components  
- 📋 Clipboard support and unit swapping
